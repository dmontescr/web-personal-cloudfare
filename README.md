# Daniel Montes Cruz - Portfolio Personal

Bienvenido al repositorio de mi página web personal. Este proyecto sirve como un portfolio interactivo y moderno para presentar mi perfil profesional, experiencia, habilidades y proyectos.

## 🚀 Sobre el Proyecto

Este portfolio ha sido diseñado con un enfoque profesional y premium, utilizando un tema oscuro con efectos de "glassmorphism", tipografías modernas y animaciones fluidas para garantizar la mejor experiencia de usuario. Refleja mi evolución profesional desde la Ingeniería Electromecánica y la gestión de proyectos hacia el Desarrollo Full Stack.

## ✨ Características Principales

*   **Diseño Responsive:** Adaptable a dispositivos móviles, tablets y escritorio.
*   **Tema Premium:** Estilo oscuro moderno con gradientes y tarjetas translúcidas (Glassmorphism).
*   **Animaciones Dinámicas:** Efectos visuales al hacer scroll, hover states interactivos y menús responsivos.
*   **Descarga de CV:** Integración directa para visualizar y descargar el currículum vitae en formato PDF.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica.
*   **CSS3:** Variables nativas, flexbox, CSS grid, animaciones, media queries.
*   **JavaScript (Vanilla):** Lógica de interacción, menú móvil, año dinámico en el footer y observación de intersección para animaciones al hacer scroll.
*   **Font Awesome:** Iconografía para botones y enlaces sociales.
*   **Google Fonts:** Tipografías *Outfit* e *Inter*.

## 📁 Estructura de Archivos

```text

├── index.html                  # Estructura principal y contenido

```

## 💻 Cómo ejecutar el proyecto

No se requiere ningún tipo de instalación ni dependencias externas. Para visualizar la página:

1.  Clona o descarga este repositorio.
2.  Asegúrate de que tu imagen de perfil esté guardada como `profile.jpg` en la raíz de la carpeta.
3.  Abre el archivo `index.html` en tu navegador web moderno favorito (Google Chrome, Firefox, Safari, Edge).
